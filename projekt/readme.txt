Przesłany folder zawiera wiele różnych plików, 
tutaj przedstawiam ich opis:
- readme.txt
Ten plik zawiera opis plików w folderze.

- variational.pdf
Ten plik zawiera wyliczenia sformułowania
wariacyjnego i wyprowadzenie układu równań
liniowych w postaci macierzowej.

- main.cpp
Ten plik zawiera główny program napisany w języku
C++, rozwiązujący uzyskany układ macierzowy oraz
zapisujący uzyskane wartości współczynników ui
w pliku "results.csv".

- results.csv
Ten plik zawiera wyliczone współczynniki ui.
Są one zapisane w formacie .csv.

- graphs_generator.ipynb
Ten plik jupyter notebooka wczytuje dane z pliku
"results.csv" i korzystając z pomocniczych 
bibliotek pakietu Python rysuje wykres funkcji.

- example_graphs.pdf
Ten plik zawiera przykładowe wykresy funkcji,
uzyskane dla odpowiednio n=4, n=10, n=100, n=1000.
